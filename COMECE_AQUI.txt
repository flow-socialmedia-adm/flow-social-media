┌──────────────────────────────────────────────────────────────┐
│  PASSO 1: INICIAR O SISTEMA                                  │
└──────────────────────────────────────────────────────────────┘

   👉 Clique DUAS VEZES em: start.bat

   O que vai acontecer:
   ✅ Docker inicia automaticamente
   ✅ API inicia na porta 3000
   ✅ Frontend inicia na porta 5173
   ✅ Abre 2 terminais (um para API, outro para Frontend)

   ⏱️  AGUARDE ~10 SEGUNDOS para tudo iniciar!


┌──────────────────────────────────────────────────────────────┐
│  PASSO 2: ACESSAR A APLICAÇÃO                                │
└──────────────────────────────────────────────────────────────┘

   👉 Abra seu navegador e vá para:

      http://localhost:5173

   🎉 Pronto! Você já pode usar o sistema!


┌──────────────────────────────────────────────────────────────┐
│  PASSO 3 (OPCIONAL): TESTAR SE ESTÁ TUDO OK                  │
└──────────────────────────────────────────────────────────────┘

   👉 Clique DUAS VEZES em: test-auto.bat

   Vai testar:
   ✅ Docker rodando
   ✅ API funcionando
   ✅ Frontend carregando
   ✅ Sistema de autenticação


┌──────────────────────────────────────────────────────────────┐
│  PARAR O SISTEMA (Quando terminar de usar)                   │
└──────────────────────────────────────────────────────────────┘

   👉 Clique DUAS VEZES em: stop.bat

   Vai parar:
   ✅ Docker
   ✅ API
   ✅ Frontend
